# DocIQ Backend

This is the backend package for DocIQ.

The full project documentation lives in the repository root `README.md`.

Backend contents:

- FastAPI application
- SQLAlchemy models and Alembic migrations
- Celery worker integration
- Storage, auth, RBAC, audit, and observability wiring
- Domain modules for documents, extraction, validation, search, workflows, feedback, analytics, and admin
