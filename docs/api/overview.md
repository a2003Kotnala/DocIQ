# API Overview

The REST API is versioned under `/api/v1` and provides:

- Authentication and refresh token flows
- Secure document upload URL generation and upload confirmation
- Document detail, extraction, validation, review, and approval endpoints
- Search and Q&A endpoints with tenant-aware access filtering
- Workflow definition and run endpoints
- Feedback, analytics, and admin surfaces

OpenAPI docs are served by FastAPI at `/docs` and `/openapi.json`.
